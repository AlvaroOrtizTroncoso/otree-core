To create a room, add to your settings.py a setting ROOMS.

ROOMS should be a list of dictionaries; each dictionary defines the configuration of a room.

For example:


    ROOMS = [
        {
            'name': 'ec101',
            'display_name': 'Econ 101',
            'participant_label_file': 'econ101.txt',
        },
        {
            'name': 'ec102',
            'display_name': 'Econ 102',
            'participant_label_file': 'econ102.txt',
        },
    ]

Here are the available keys:

  • name: internal name (required)
  • display_name: display name (required)
  • participant_label_file: a path to a text file with the "guest list" for this room. Should have one participant label per line.
  • use_secure_urls: whether oTree should add hashes to URLs for extra security.
